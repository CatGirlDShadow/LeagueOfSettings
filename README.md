# LeagueOfSettings
Tool that allows you to change your LoL language

## Available languages
 - English (en_US)
 - English (GB) (en_GB)
 - English (AU) (en_AU)
 - Simplified Chinese (zh_CN)
 - Traditional Chinese (zh_TW)
 - German (de_DE)
 - French (fr_FR)
 - Greek (el_GR)
 - Italian (it_IT)
 - Japanese (ja_JP)
 - Korean (ko_KR)
 - Polish (pl_PL)
 - Portuguese (pt_PT)
 - Romanian (ro_RO)
 - Russian (ru_RU)
 - Spanish (es_ES)
 - Turkish (tr_TR)
